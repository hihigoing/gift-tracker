# PWA 图标说明

## 🎨 当前状态

已生成 SVG 图标：`public/icon.svg`

## 📱 如何获取 PNG 图标

### 方法 1：在线转换（推荐）

1. 访问 https://cloudconvert.com/svg-to-png
2. 上传 `public/icon.svg`
3. 设置尺寸为 192x192，下载为 `icon-192.png`
4. 再次转换，设置尺寸为 512x512，下载为 `icon-512.png`
5. 将两个文件放到 `public/` 目录

### 方法 2：使用 macOS 预览

1. 双击打开 `public/icon.svg`
2. 文件 → 导出
3. 格式选择 PNG
4. 尺寸输入 192x192（或 512x512）
5. 保存

### 方法 3：使用网站生成

访问 https://realfavicongenerator.net/
- 上传任意图片
- 自动生成所有尺寸的 PWA 图标
- 下载并解压到 `public/` 目录

## ⚠️ 没有图标会怎样？

- ✅ PWA 功能完全正常
- ✅ 可以安装到桌面
- ⚠️ 桌面显示默认图标或首字母
