## 2️⃣ 安装环境

### 安装 Node.js

```bash
# Ubuntu/Debian
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 或者使用 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
source ~/.bashrc
nvm install 18
```

### 安装 MySQL

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install -y mysql-server
sudo systemctl start mysql
sudo systemctl enable mysql

# 初始化 MySQL 安全配置
sudo mysql_secure_installation
```

---

## 3️⃣ 配置 MySQL

```bash
# 登录 MySQL
sudo mysql -u root -p

# 创建数据库和用户
CREATE DATABASE gift_tracker DEFAULT CHARACTER SET utf8mb4;
CREATE USER 'gift_user'@'localhost' IDENTIFIED BY '你的强密码';
GRANT ALL PRIVILEGES ON gift_tracker.* TO 'gift_user'@'localhost';
FLUSH PRIVILEGES;
EXIT;
```

---

## 4️⃣ 部署代码

```bash
# 克隆代码
cd /opt
git clone https://github.com/hihigoing/gift-tracker.git
cd gift-tracker/gift-tracker-final

# 安装依赖
npm install

# 创建 .env 文件
cat > .env << EOF
DB_HOST=localhost
DB_PORT=3306
DB_USER=gift_user
DB_PASSWORD=你的强密码
DB_NAME=gift_tracker
PORT=3000
EOF

# 初始化数据库
npm run init-db
```

---

## 5️⃣ 配置防火墙

```bash
# Ubuntu UFW
sudo ufw allow 3000/tcp
sudo ufw reload

# 或者 CentOS/RHEL
sudo firewall-cmd --zone=public --add-port=3000/tcp --permanent
sudo firewall-cmd --reload

# 阿里云/腾讯云需要在控制台安全组添加入站规则
# 协议：TCP
# 端口：3000
# 授权对象：0.0.0.0/0
```

---

## 6️⃣ 启动服务

### 方式 1：使用 PM2（推荐，生产环境）

```bash
# 安装 PM2
npm install -g pm2

# 启动应用
pm2 start server.js --name gift-tracker

# 开机自启
pm2 startup
pm2 save

# 查看状态
pm2 status
pm2 logs
```

### 方式 2：直接运行（测试用）

```bash
npm start
```

## ⚠️ 安全建议

1. **修改默认密码**
   - MySQL root 密码
   - 数据库用户密码

2. **配置 HTTPS（可选）**
   - 使用 Nginx 反向代理
   - 申请 Let's Encrypt 证书

3. **限制数据库访问**
   - 只允许 localhost 访问
   - 不要开放 3306 端口到公网

4. **定期备份**
   ```bash
   mysqldump -u gift_user -p gift_tracker > backup.sql
   ```

## 📞 故障排查

**Q: 无法访问 API？**
```bash
# 检查服务是否运行
pm2 status

# 检查防火墙
sudo ufw status

# 检查端口监听
netstat -tlnp | grep 3000
```

**Q: 跨域错误？**
- 检查 CORS 配置是否正确
- 浏览器控制台查看详细错误

**Q: 数据库连接失败？**
- 检查 .env 配置
- 确认 MySQL 服务运行
- 检查数据库用户权限
